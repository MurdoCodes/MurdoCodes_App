# MurdoCodes App

MurdoCodes App is for personal use
Built out of
    - React Native
    - NodeJS
    - Docker

## Plan
1. Register
2. Login/Logout
3. Profile
	- Upload Photo
	- Edit Personal Details
		* Name
		* Username
		* Date of Birth
		* Phone/Mobile Number
		* Address

> APP Pages
	- Home
	- Registration
	- Login
	- Logout
	- Settings
		* Profile


> Services
	- Personal Time Management app
		* user can set alarms and dates with title and description
	- List available jobs from onlinejobs.ph

## Installation
> Addition Package Installed
	- expo-font
	- axios
	- react-native-dotenv
